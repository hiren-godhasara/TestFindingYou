
# Finding You



##  Prerequisites

Before running the application, ensure you have the following installed on your machine:

- Node.js

## Installation

1. Clone the repository:


```bash
   git clone <repository-url>
```

2.  Install dependencies:

```bash
cd TestFindingYou
npm install
```

3. Start the application:

```bash
npm run start:dev
```

## Technologies Used


* Node.js
* TypeScript
* Nest.js
